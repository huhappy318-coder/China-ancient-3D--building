# REMIND

## 项目定位

这是一个基于 `Vite + React + Tailwind CSS + React Three Fiber` 的中国古建筑自由搭建网页原型。

当前已经演进到 V3，重点能力包括：

- 多对象搭建与编辑
- 单选 / 多选
- 撤销 / 重做
- 模板快速生成
- 图纸辅助与中轴引导
- 本地保存 / 恢复
- JSON 导入 / 导出
- 场景截图导出
- 基于规则的完成度反馈

## 本地运行

```bash
npm install
npm run dev
```

构建验证：

```bash
npm run build
```

## 当前关键文件

- `src/App.jsx`
  页面主状态、工具栏逻辑、撤销重做、导入导出、多选与模板接入都在这里统一编排。

- `src/components/scene/`
  Three 场景与对象渲染层。

- `src/components/panels/`
  左侧各功能面板，包括构件、模板、图纸、属性等。

- `src/utils/`
  历史栈、模板、评分、导入导出、镜像复制、阵列复制等工具逻辑。

## 继续开发时优先注意

1. 不要推翻当前“网页产品感”，保持主场景优先。
2. 新功能尽量继续增量开发，不要把工程做成重型 DCC 工具。
3. 对象编辑逻辑如果继续变复杂，优先考虑把 `App.jsx` 中的场景状态抽成 hooks。
4. 导出截图依赖当前 `canvas`，如后续引入多画布要同步调整。
5. 撤销 / 重做目前基于快照，后续若对象量继续增多，可再优化为更细粒度历史记录。

## 建议优先的下一步

- 场景内拖动对象
- 更细的古建筑构件规则
- 多对象编组
- 更真实的屋顶 / 梁架参数化
- 更强的模板系统与作品展示页
