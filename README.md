# 中国古建筑自由搭建网页

一个面向网页端的中国古建筑轻量搭建原型项目。

它不是传统后台页面，也不是重型 DCC 软件，而是一个强调“沉浸感、产品感、可编辑性”的 3D 古建筑网页编辑器：用户进入页面后，直接面对搭建场景，通过左侧窄边框与滑出面板完成构件放置、对象编辑、图纸辅助、天气光影切换与方案导出。

---

## 项目定位

这个项目的目标不是一步到位做成专业建模工具，而是逐步演进出一个更适合展示、体验和轻搭建的古建筑网页产品。

当前版本已经演进到 **V3**，重点增强了：

- 多对象搭建与编辑
- 单选 / 多选
- 撤销 / 重做
- 模板化快速生成
- 中轴与图纸辅助
- 本地保存 / 恢复
- JSON 导入 / 导出
- 场景截图导出
- 基于规则的完成度反馈

## 当前能力

### 1. 场景搭建

- 支持在 3D 场景中新增多个构件对象
- 当前支持的基础构件包括：
  - 台基
  - 立柱
  - 横梁
  - 屋顶
  - 飞檐
  - 石阶
- 所有对象都基于 React Three Fiber 真实渲染

### 2. 编辑能力

- 点击对象可单选
- 支持 `Shift + 点击` 多选
- 属性面板支持：
  - 位置编辑
  - 旋转编辑
  - 缩放编辑
  - 显示 / 隐藏
  - 删除
  - 复制
- 多选模式下支持批量移动、批量缩放、批量显隐、批量删除

### 3. 规则化辅助

- 场景内有网格与中轴线
- 对象位置支持吸附
- 支持镜像复制
- 支持阵列复制
- 支持图纸叠加与推荐位置提示

### 4. 模板系统

支持快速生成以下预设方案：

- 对称门楼
- 四柱亭
- 单檐殿阁

模板生成后仍可继续编辑。

### 5. 产品化能力

- 顶部轻工具栏
- 天气 / 光影切换
- 俯视 / 透视 / 图纸叠加 / 自由漫游视角
- 本地保存与恢复
- 导出 JSON
- 导入 JSON
- 导出场景截图
- 搭建完成度与建议反馈

## 目录结构

```txt
src/
  components/
    layout/         # 左侧边框、顶部工具栏、统计卡、视角按钮
    panels/         # 构件、模板、图纸、天气、光影、属性等面板
    scene/          # Three 场景、对象渲染、图纸参考、引导层
  data/             # 导航、构件目录、步骤文案
  utils/            # 历史栈、模板、导入导出、评分、镜像、阵列等工具
  App.jsx           # 页面主编排与状态管理
  main.jsx
  index.css
```

## 交互说明

### 左侧结构

- 左侧第一层是窄边框导航
- 点击后从左侧展开对应功能面板
- 支持隐藏整个左侧区域，并通过悬浮按钮重新展开

### 核心工作流

1. 从构件库添加对象，或直接套用模板
2. 点击场景对象进行选择
3. 在属性面板中调整位置、旋转、缩放
4. 配合图纸辅助与中轴线进行规整搭建
5. 使用天气、光影和视角增强展示效果
6. 保存、导出或截图输出成果

## 撤销与重做

当前版本已支持：

- 顶部按钮撤销 / 重做
- 快捷键：
  - `Ctrl/Cmd + Z` 撤销
  - `Ctrl/Cmd + Shift + Z` 重做

历史记录当前采用快照式实现，适合原型阶段快速稳定落地。

## 导入导出

支持以下输出方式：

- 导出 JSON
- 导入 JSON
- 导出当前场景截图

同时项目也支持将当前场景状态保存在本地 `localStorage` 中，刷新后自动恢复。
